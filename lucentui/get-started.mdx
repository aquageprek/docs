## ðŸ“¦ Option 2: Install via PTB (.paytheme)

Using [PTB (Paymenter ToolBox)](https://github.com/QKing-Official/PTB) is the **recommended** way to install and manage your theme.

<Steps>
<Step title="Install PTB CLI">
Run this script to install PTB globally:
```bash
curl -fsSL https://raw.githubusercontent.com/QKing-Official/PTB/refs/heads/main/installer | bash
```
</Step>

<Step title="Move the .paytheme file">
Upload the `lucentui.paytheme` file to your hosted Paymenter server (any location is fine).
</Step>

<Step title="Import using PTB">
Import the theme with this command:
```bash
ptb import -t lucentui.paytheme
```
</Step>

<Step title="Activate LucentUI">
Go to your **Admin â†’ Settings â†’ Theme**, and select `lucentui`.
</Step>
</Steps>
