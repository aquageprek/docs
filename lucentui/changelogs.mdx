<Update label="2025-07-26" description="v1.2.2" tags={["Bug Fix", "Enhancement", "Extension"]}>
## 📦 Version 1.2.2 — UI Improvements, Bug Fixes & Extension Support

### ✨ What's New & Improved

- **Refined Look & Unified Layout** — Borders have been softened, transparency has been polished, and spacing is now more consistent across the board
- **🧾 Redesigned Invoice and Service Page** — Sleek, clean, and more professional to give your customers a better first impression
- **⚙️ PterodactylPlus Extension Support** — LucentUI now works out-of-the-box with PterodactylPlus!
- **🔐 Password Strength Indicator** — Get real-time feedback while typing your password during registration
- **🌍 Locale-Based Time & Date Display** — Dashboard now shows time and date in your selected language and format
- **🔧 Added More Customization Options** — Fine-tune your dashboard with additional layout and design settings such as Custom CSS and HTML Addon

### 🐛 Bug Fixes

- Minor animation, layout, and UI issues have been squashed
- Improved overall stability and performance

  ```bash
  ptb import -t lucentui.paytheme  
  ```

- Performance optimizations
- Refined UI elements
- Better theme consistency
